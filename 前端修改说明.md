# 前端修改说明

## 已自动完成的修改

桌面文件夹 `VIPChinaGuide SEO and Safe 7.14 up` 里的 `booking.html` 已修改：

### 1. 替换邮件发送逻辑
- 删除了 FormSubmit.co 的旧代码
- 改为调用你自己的 Vercel 后端 API
- 还没部署时显示提示，不影响使用

### 2. 更新 CSP 安全头
- `connect-src` 中把 `https://formsubmit.co` 换成了 `https://*.vercel.app`
- `form-action` 中去掉了 `https://formsubmit.co`

## 你还需要手动做的（部署后）

### 步骤 A: 替换 API 地址
打开 `booking.html`，搜索 `YOUR_VERCEL_API_URL`，改成你的实际地址：

```javascript
const API_URL = 'https://vcg-backend-xxx.vercel.app/api/send-email';
```

### 步骤 B: 精确 CSP 域名（可选但推荐）
如果 Vercel 分配的是 `https://vcg-backend-abc123.vercel.app`，建议把 CSP 中的：
```
https://*.vercel.app
```
替换为具体域名：
```
https://vcg-backend-abc123.vercel.app
```

这样更安全，防止其他恶意 vercel.app 域名访问你的 API。
